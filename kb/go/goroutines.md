# Горутины в Go

## Что такое горутина?
Горутина — это легковесный поток выполнения, управляемый рантаймом Go. Начальный стек занимает ~2 КБ и растёт динамически.

Пример запуска:
```go
func sayHello() {
    fmt.Println("Привет из горутины!")
}

func main() {
    go sayHello()
    time.Sleep(100 * time.Millisecond)
}
```

## Ключевые особенности
| Характеристика | Описание |
|---------------|----------|
| Легковесность | ~2 КБ начального стека, масштабируется до миллионов горутин |
| Планировщик | M:N модель (M горутин на N системных потоков) |
| Запуск | Ключевое слово `go` перед вызовом функции |
| Жизненный цикл | Завершается вместе с `main()` или при панике без recover |

## Паттерны использования
Параллельная обработка:
```go
urls := []string{"a.com", "b.com", "c.com"}
for _, url := range urls {
    go fetchURL(url)
}
// ⚠️ В продакшене всегда используйте sync.WaitGroup или channels
```

## Частые ошибки
❌ Забыли дождаться завершения:
```go
for i := 0; i < 10; i++ {
    go doWork(i)
}
// main() завершается раньше, чем горутины отработают
```

✅ Корректное ожидание:
```go
var wg sync.WaitGroup
for i := 0; i < 10; i++ {
    wg.Add(1)
    go func(id int) {
        defer wg.Done()
        doWork(id)
    }(i)
}
wg.Wait()
```
