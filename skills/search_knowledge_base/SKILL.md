# Навык: Поиск в базе знаний

## Когда использовать

- Пользователь спрашивает о конкретной технологии или концепции
- Нужны факты, примеры кода, определения из учебных материалов
- Запрос содержит термины курса: `defer`, `pod`, `channel`, `interface`

## Алгоритм выполнения

1. Если `topic` не равен `"other"` → применить фильтр по тегам в Qdrant
2. Векторизовать `query` через модель `nomic-embed-text`
3. Выполнить гибридный поиск (vector + BM25) в коллекции `course_kb`
4. Отфильтровать результаты по `score > 0.3` и `level <= requested_level`
5. Вернуть топ-3 чанка с метаданными

## Обработка ошибок

- Если найдено меньше 2 чанков → вернуть статус `insufficient`
- Если Qdrant недоступен → логировать ошибку, вернуть пустой список
- Если эмбеддинг не сгенерировался → повторить запрос 1 раз

## Пример вызова

Входные данные:

```
{
"query": "Как работает defer в Go?",
"topic": "go",
"level": "beginner"
}
```

Выходные данные:

```
{
"chunks": [
{
"text": "defer откладывает выполнение функции...",
"source": "defer.md",
"score": 0.87,
"tags": ["go", "defer", "cleanup"]
}
],
"count": 1
}
```
