# Навык: Оценка качества контекста

## Когда использовать
- После поиска в базе знаний, перед генерацией ответа
- Когда нужно решить: отвечать по найденному или запросить уточнение

## Критерии достаточности

Достаточно (sufficient), если:
- Найдено >= 2 чанка И
- Средняя релевантность >= 0.6 И
- Хотя бы один чанк содержит прямой ответ на вопрос

Недостаточно (insufficient), если:
- Чанков < 2 ИЛИ
- Средняя релевантность < 0.6 ИЛИ
- Все чанки касаются смежной, но не целевой темы

## Алгоритм (см. scripts/quality_check.py)
1. Вычислить среднюю релевантность: avg_score = mean(chunk.score for chunk in chunks)
2. Проверить наличие ключевых слов из `query` в текстах чанков
3. Применить пороговую логику (см. критерии выше)
4. Вернуть статус + обоснование

## Пример

Входные данные:
```
{
"chunks": [
{"text": "defer откладывает...", "score": 0.87},
{"text": "LIFO порядок...", "score": 0.72}
],
"query": "Как работает defer?"
}
```
Выходные данные:
```
{
"status": "sufficient",
"reason": "2 чанка, avg_score=0.80, содержат прямой ответ",
"avg_score": 0.80
}
```
